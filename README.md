ee4352-cmos-vlsi
================

CMOS VLSI Design with Dr. Aslan Fall 2013 ; Final project 4-bit ALU.

The goal is to design a 4-bit ALU that can handle the following:

* Addition
* Subtraction
* Inversion
* Comparison
* Shifting
* Appropriate overflow, sign, and other flags.
